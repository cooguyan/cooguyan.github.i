太极框架实在是太好用了，强烈安利

Step1:安装Magisk

1.1 解除BL(Boy’s Love)锁：

进入申请解锁页面按照提示操作即可(ง •_•)ง

1.2 安装第三方Recovery

在电脑上进行下面的操作：

  1. 下载奇兔刷机并安装
  2. 长按 音量下+关机键 直到屏幕出现一个修机器人的兔子🐇为止
  3. 此时打开奇兔刷机，并将手机用数据线连接电脑
  4. 进入“实用工具”，找到“刷入Recovery”选项并打开
  5. 选择“匹配。。。刷入” ，等待它自动完成即可（如果它失败了，只能自己找了hhhh）

假设奇兔刷机圆满完成任务,那么此时你的手机几乎没有异样，可以正常开机，数据线也可以拔掉了，后续的操作不关电脑的事情。

如果它自动进入Recovery界面，那么长按关机就可以退出了。
另外，手动进入Recovery模式的方式是 长按 音量上+关机键

此处不得不说一下这个第三方recovery的强大之处。它强就强在它可以将手机自动解密！尽管这在我刷xposed变砖的时候帮了我大忙：我当时文件没有备份，不知道怎么在不开机的情况下转移数据。查了一些资料说是出于保护手机的需要，手机内容都是加密的。
但这也意味着如果别人捡到我的手机，按照这个教程就可以在不开机的情况下读取我手机里的数据，保护作用形同虚设。

1.3 刷入Magisk

开机后在手机上进行下面的操作：

  1. 在手机上下载Magisk，它是个zip包，要记住它下载到哪里了。
  2. 手动进入Recovery模式
  3. 选择第一个选项 ：安装
  4. 找到你下载的Magisk.zip包，勾选它安装。
  5. 完成后重启


图中的挂载功能可以把存储器挂载到电脑上进行读写数据
ok，至此Magisk已经安装到手机中了，手机中此时已经多了一个脸谱图标的应用

找不到下载地方？ 浏览器下载的文件都是保存在Download文件夹中

Step2:安装太极：

在手机上进行以下操作：

  1. 下载magisk-taichi.zip,（链接是个看起来不太正规的网盘，但是能用）
  2. 打开Step1安装好的APP，如果没有可以手动下载
  3. 在侧边栏选择模块
  4. 点击屏幕下方的加号，找到下载好的magisk-tachi压缩包进行安装。

到这里太极就安装完成，手机上会多一个太极图标的应用。如果Magisk安装成功，那么打开后的太极左上角就是 太极.阳，不然就是太极.阴

Step3:下载模块并使用

官方太极使用说明：
太极支持的模块列表（有功能介绍）

  1. 打开太极，添加你需要使用 Xposed 模块的 APP，如QQ到太极中。
  2. 进入模块管理，勾选上你需要使用的 Xposed 模块，如 去你大爷的内置浏览器。
  3. 强制停止你需要观察效果的APP，如上面的QQ。
  4. 打开QQ观察效果。
